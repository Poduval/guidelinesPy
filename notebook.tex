    PYTHON STATEMENTS

Learning about statements will allow you to be able to read other
languages more easily in the future. If already know a different
language this will rapidly accelerate your understanding of Python.

\textbf{Indentation:} It is important to keep a good understanding of
how indentation works in Python to maintain the structure and order of
your code. Python is heavily driven by code indentation and whitespace,
this means that code readability is a core part of the design of the
Python language. e.g.

\begin{verbatim}
if x:
    if y:
        code-statement
else:
    another-code-statement
\end{verbatim}

    \section{if, elif, else Statements}\label{if-elif-else-statements}

\texttt{if} statements in Python allows us to tell the computer:

"\emph{Hey \texttt{if} this case happens, perform some action}".

We can then expand the idea further with \texttt{elif} and \texttt{else}
statements, which allow us to tell the computer:

"\emph{Hey \texttt{if} this case happens, perform some action.
\texttt{Else,\ if} another case happens, perform some other action.
\texttt{Else,\ if} none of the above cases happened, perform this
action}"

code format:

\begin{verbatim}
if condition_01:
    perform action_01
elif condition_02:
    perform action_02
else: 
    perform action_03
\end{verbatim}

    \section{for Loops}\label{for-loops}

A \texttt{for} loop acts as an iterator in Python; it goes through items
that are in a \textbf{sequence} or any other iterable item.

code format:

\begin{verbatim}
for item in object:
    statements to do stuff
    
\end{verbatim}

The variable name used for the \texttt{item} is completely up to you.
This \texttt{item} name can then be referenced inside your loop.

    \section{while Loops}\label{while-loops}

A \texttt{while} statement will repeatedly execute a single statement or
group of statements as long as the condition is true. The general format
of a while loop is:

\begin{verbatim}
while condition:
    code statements
else:
    some other code statements
    
    
\end{verbatim}

Caution !!! DO NOT RUN THIS CODE

\begin{verbatim}
while True:
    print("I'm stuck in an infinite loop!")
\end{verbatim}

    \subsection{break, continue, pass}\label{break-continue-pass}

\texttt{break}, \texttt{continue}, and \texttt{pass} statements in our
loops are additional functionality for various cases.

\begin{itemize}
\tightlist
\item
  \texttt{break}: Breaks out of the current closest enclosing loop.
\item
  \texttt{continue}: Goes to the top of the closest enclosing loop.
\item
  \texttt{pass}: Does nothing at all.
\end{itemize}

\texttt{break} and \texttt{continue} statements can appear anywhere
inside the loop's body, but usually put use them with an \texttt{if}
statement to perform an action based on some condition.
